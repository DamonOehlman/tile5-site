Displayed below are the roadmapped releases of Tile5 for the next few months.

Items from each of the releases are fed from the Tile5 Github Repository so if there is something in particular you are chasing then please consider raising the request in the Issue Tracker and we look at when such a request could be implemented.

0.9.5

This will be the next release of Tile5. It focuses on adding the ability to drag markers and path and also support hover events for desktop browsers. Additionally, we will be starting to test Internet Explorer compatibility via Flash Canvas in this release.

    0.9.5.1

    A maintenance release of 0.9.5 that will look to restore some of the more refined zoom behaviour that was found in the 0.9.3 release back to the 0.9.5 and beyond releases.
