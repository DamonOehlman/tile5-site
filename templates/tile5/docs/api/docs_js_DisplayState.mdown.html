DisplayState

The DisplayState member of the T5.Graphics module is an enumerated type (well, kind of) that is used to flag the current display state of a Graphics.View object.

Valid values of the display state are:

  • NONE: 0
  • ACTIVE: 1
  • ANIMATING: 4
  • PAN: 8
  • PINCHZOOM: 16
  • FREEZE: 128
