Tile5 is an opensource HTML5 mapping javascript library that provides developers the ability to integrate existing mapping platforms to provide a rich HTML5 experience.

In addition to providing a great API for mapping, Tile5 also excels at building more generic tiling application interfaces.

Device Compatibility

Tile5 is targeted at both mobile and desktop browsers that support HTML5. Our long-term goal is to achieve compatibility with all mobile devices considered "Class A" on the jQuery Mobile GBS chart, as well as modern desktop browsers (including IE once Microsoft get their act together).

Tile5 also plays nice with a variety of Mobile Web UI Frameworks.

News and Updates

To keep you up-to-date with the various changes that are going on with the library, check out our news page.

Get the Source

The source for Tile5 is maintained on Github, and released under the MIT license.
