An open mapping library for mobile and desktop.

Rich HTML5 maps. One codebase, many devices.

Tile5 is an opensource HTML5 mapping Javascript library. It lets you use your favourite mapping platform to provide a rich web experience on both mobiles and desktops.

Why use Tile5?

Developing mapping apps to support multiple mobile devices and desktops is painful. Maintaining multiple codesets for different devices, tied to one set of mapping data, is no fun.

Tile 5 offers:

  • One codebase to maintain
  • Multiple mapping providers
  • Mapping in HTML5 that feels like a native app
  • Runs where HTML5 does, on both mobile and desktop browsers
  • Compatible with a variety of Mobile Web UI Frameworks

In addition to providing a great API for mapping, Tile5 also excels at building more generic tiling application interfaces.
