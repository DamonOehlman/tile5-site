We are currently working on a number of demonstration apps for Tile5 to showcase its capabilities. Below are two of the demos currently being constructed:

HTML5 Mapping Demo Console

The HTML5 mapping demo console showcases some of the basic mapping operations that are provided by Tile5 along with some code samples of how the console works.

HTML5 Mapping Demo Console

HTML5 Mapping Sandbox

With things starting to progress here with Tile5 all demos on this site are now using the latest stable version of the Tile5 library. To make sure we can keep things exciting though, we will be developing demos with bleeding edge changes and possibly even community forks in the sandbox.

HTML5 Mapping Sandbox

Flickr Photo Explorer

The Flickr photo explorer demonstrates that Tile5 is not just about mapping. Using the tile based grid to search for and display flickr photos, you really can look through and process much more information than would be possible through a traditional display.

Flickr Photo Explorer

Endless Mural Mashup

Once the base code for the Photo Explorer was written, making a demo that pulled in the images from the awesome Endless Mural project was a relatively simple task.

Endless Mural Mashup
