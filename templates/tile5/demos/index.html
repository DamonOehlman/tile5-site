We are currently working on a number of demonstration apps for Tile5 to showcase its capabilities. Below are two of the demos currently being constructed:

HTML5 Mapping Demo Console

The HTML5 mapping demo console showcases some of the basic mapping operations that are provided by Tile5 along with some code samples of how the console works.

Flickr Photogrid

The Flickr photgrid demonstrates that Tile5 is not just about mapping. Using the tile based grid to search for and display flickr photos, you really can look through and process much more information than would be possible through a traditional display.

Have an Idea?

If you have an idea for what could be built with Tile5 and would like to see a proof of concept, or are interested in contributing a demo then please get in touch via Twitter or by emailing tile5 -at- sidelab.com.
